🛠️ Outils de Hacking Avancés

Outils essentiels de reconnaissance et scanning

🌐 Nmap (Network Mapper)

L'outil de scan de réseau le plus populaire.

  • Fonction : Scanner les ports, identifier les services
  • Commande basique : nmap -sV target.com
  • Options : -p (ports spécifiques), -A (détection agressive), -O (détection OS)
  • Utilité : Découvrir les services actifs et les vulnérabilités potentielles

🔎 Wireshark

Analyseur de trafic réseau puissant.

  • Fonction : Capturer et analyser les paquets réseau
  • Utilité : Voir le trafic en temps réel, détecter les anomalies
  • Cas d'usage : Déboguer les problèmes réseau, détecter les intrusions
  • Avantage : Interface graphique intuitive

🔑 Metasploit Framework

Framework complet pour les tests de pénétration avec des exploits.

  • Fonction : Exploiter les vulnérabilités automatiquement
  • Contient : +2000 exploits, payloads, modules de reconnaissance
  • Utilité : Tester automatiquement les vulnérabilités
  • Commande : msfconsole
  • Payloads : Meterpreter (accès interactif), reverse shells, webshells

Outils de web hacking avancés

🕷️ Burp Suite (COMPLET)

La plateforme N°1 pour tester la sécurité des applications web.

  • Modules principaux :
    • Proxy : Intercepte et modifie les requêtes HTTP/HTTPS
    • Scanner : Scan automatique des vulnérabilités
    • Repeater : Permet de rejouer les requêtes
    • Intruder : Attaques par force brute personnalisées
    • Decoder : Décode/encode les données
    • Comparer : Compare deux requêtes/réponses
    • Extender : Plugins personnalisés
  • Vulnérabilités détectées : XSS, CSRF, SQL Injection, XXE, SSRF, etc.
  • Versions : Community (gratuit) et Professional (payant)

🧪 OWASP ZAP (Zed Attack Proxy)

Outil gratuit et open source pour scanner les applications web.

  • Fonction : Tester les vulnérabilités web
  • Avantage : Gratuit et open source
  • Modules : Proxy, Scanner, Fuzzer, Spiderquad
  • Utilité : Alternative à Burp Suite

🔧 SQLmap

Outil automatisé pour détecter et exploiter les injections SQL.

  • Fonction : Automatiser l'exploitation des vulnérabilités SQL
  • Commande : sqlmap -u "URL" --dbs
  • Utilité : Accéder aux bases de données
  • Avantage : Très automatisé et efficace

📡 Nikto

Scanner web pour détecter les vulnérabilités communes.

  • Fonction : Scan les serveurs web
  • Détecte : Fichiers dangereux, versions obsolètes, configurations faibles
  • Commande : nikto -h target.com

🎯 Dirbuster / Gobuster

Outils pour découvrir les répertoires et fichiers cachés.

  • Dirbuster : Version graphique
  • Gobuster : Version CLI ultra rapide
  • Commande : gobuster dir -u target.com -w wordlist.txt
  • Utilité : Trouver les pages admin, API cachées, backups

Outils de reconnaissance OSINT avancés

🕵️ Shodan

Moteur de recherche pour les appareils connectés à Internet.

  • Fonction : Trouver les appareils, caméras, serveurs exposés
  • Utilité : Reconnaissance passive
  • Site : shodan.io
  • Exemples : Caméras IP, routeurs, serveurs web

🌍 TheHarvester

Outil pour collecter des informations publiques.

  • Fonction : Chercher des emails, sous-domaines, adresses IP
  • Sources : Google, Bing, LinkedIn, Twitter, Censys
  • Utilité : Reconnaissance initiale
  • Commande : theHarvester -d domain.com -b all

🔍 Sublist3r

Outil pour découvrir les sous-domaines.

  • Fonction : Énumérer tous les sous-domaines
  • Sources : Google, Censys, Shodan, VirusTotal
  • Commande : sublist3r -d domain.com

🗺️ Maltego

Plateforme de transformation d'informations avec visualisation.

  • Fonction : Mapper les relations entre les données
  • Utilité : Analyse visuelle des données
  • Transforms : +200 transformations disponibles

Outils de brute force et password cracking

🔐 Hashcat

GPU-accelerated password recovery tool ultra rapide.

  • Fonction : Cracker les hashes de mots de passe
  • Utilise : La puissance GPU pour accélérer (10-100x plus rapide)
  • Supporte : 300+ formats de hash (MD5, SHA, bcrypt, NTLM, etc.)
  • Modes : Dictionary, Brute-force, Hybrid, Mask

🔑 John the Ripper

Outil classique et efficace de crackage de mots de passe.

  • Fonction : Cracker les mots de passe
  • Commande : john hashes.txt
  • Utilité : Tester la force des mots de passe

🎯 Hydra

Outil d'attaque par force brute multiprotocoles.

  • Fonction : Tester les identifiants par force brute
  • Protocoles : SSH, HTTP, FTP, IMAP, SMTP, RDP, MySQL, etc.
  • Commande : hydra -l user -P passwords.txt target.com ssh
  • Avantage : Parallélisation rapide

Outils WiFi et réseau

📡 Aircrack-ng

Suite complète pour l'audit des réseaux WiFi.

  • Airodump-ng : Capture et énumère les réseaux WiFi
  • Aireplay-ng : Génère du trafic réseau
  • Aircrack-ng : Craque les mots de passe WiFi
  • Utilité : Audit complet des réseaux sans fil

🔌 Ettercap

Outil de sniffing et MITM (Man-in-the-Middle).

  • Fonction : Capture des données, ARP spoofing, DNS spoofing
  • Utilité : Tests de sécurité réseau

Outils post-exploitation avancés

🐚 Mimikatz

Outil pour extraire les credentials de Windows.

  • Fonction : Dump des mots de passe, des tickets Kerberos
  • Utilité : Post-exploitation sur Windows
  • Avantage : Très efficace pour escalade de privilèges

🔌 Empire

Framework de post-exploitation basé sur PowerShell.

  • Fonction : Command & Control, post-exploitation
  • Utilité : Maintenir l'accès
  • Agent : S'exécute en mémoire, difficile à détecter

🎭 BeEF (Browser Exploitation Framework)

Framework pour l'exploitation de navigateurs web.

  • Fonction : Exploiter les navigateurs compromis
  • Utilité : Contrôler des navigateurs, voler des données
  • Modules : Keylogging, screenshot, redirection, etc.

🔓 Cobalt Strike

Framework professionnel de post-exploitation et C2 (payant).

  • Fonction : Red team operations avancées
  • Features : Beacons, pivoting, lateral movement
  • Note : Outil professionnel (très cher)

Outils d'énumération avancée

🔍 Enum4linux

Énumération complète des systèmes Linux/Samba.

  • Fonction : Extraire les informations des serveurs
  • Détecte : Utilisateurs, partages, services

💻 BloodHound

Outil de mapping des domaines Active Directory.

  • Fonction : Visualiser les chemins d'attaque dans Active Directory
  • Utilité : Identifier les escalades de privilèges possibles

Distributions Linux pour le hacking

🐉 Kali Linux

Distribution Linux complète pour les tests de pénétration.

  • Contient : +600 outils de hacking pré-installés
  • Avantage : Prêt à l'emploi, tous les outils
  • Utilité : La référence du hacking

🔴 Parrot Security OS

Alternative légère et rapide à Kali Linux.

  • Avantage : Plus léger, plus rapide
  • Contient : Des outils similaires à Kali

🛡️ BlackArch Linux

Distribution basée sur Arch Linux avec 2000+ outils.

  • Avantage : Très flexible, rolling release
  • Utilité : Pour utilisateurs avancés

Outils de virtualisation

  • VirtualBox : Gratuit, pratique pour débuter
  • VMware Workstation : Professionnel, plus cher
  • Hyper-V : Intégré à Windows Pro
  • KVM : Pour Linux

Frameworks et scripts

🐍 Exploit-DB

Base de données d'exploits publics.

  • Contient : +60 000 exploits
  • Site : exploit-db.com

📦 SearchSploit

Outil CLI pour interroger Exploit-DB.

  • Commande : searchsploit apache 2.4

Outils de documentation

  • Notion : Organiser les findings
  • CherryTree : Prise de notes structurée
  • Dradis : Plateforme de reporting

⚠️ Bonnes pratiques pour l'utilisation des outils

  • ✅ Utilisez les outils UNIQUEMENT sur des systèmes autorisés
  • ✅ Documentez ce que vous faites
  • ✅ Apprenez comment fonctionne chaque outil
  • ✅ Testez d'abord dans un environnement isolé
  • ✅ Respectez les lois locales
  • ✅ Signalez les failles découvertes responsablement
  • ✅ Mettez les outils à jour régulièrement
