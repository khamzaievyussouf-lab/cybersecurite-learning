⚙️ Techniques de Hacking

🎯 Social Engineering (Ingénierie sociale)

Manipulation psychologique pour obtenir des informations confidentielles.

Tactiques courantes :

  • Pretexting : Se faire passer pour quelqu'un d'autre (IT support, etc.)
  • Phishing : Envoyer des emails trompeurs
  • Baiting : Laisser un USB infecté dans un parking
  • Tailgating : Suivre quelqu'un à travers une porte sécurisée
  • Quid pro quo : Proposer un service en échange d'informations

🔐 Attaques par brute force

Essayer toutes les combinaisons possibles de mots de passe.

Types :

  • Dictionary attack : Utilise une liste de mots courants
  • Brute force : Essaie toutes les combinaisons possibles
  • Hybrid attack : Combine dictionary et modifications (ajouter des chiffres)
  • Rainbow tables : Tables pré-calculées de hashes

Défense :

  • ✅ Mots de passe forts
  • ✅ Limitation de tentatives
  • ✅ Délai entre tentatives
  • ✅ Authentification multi-facteurs

💉 Injection SQL

Injecter du code SQL dans les champs de formulaire.

Exemple :

Champ de recherche : ' OR '1'='1
Le système exécute : SELECT * FROM users WHERE name = '' OR '1'='1'

Conséquences :

  • Accès aux données de la base de données
  • Suppression ou modification de données
  • Contournement d'authentification

Défense :

  • ✅ Requêtes préparées (Prepared Statements)
  • ✅ Validation des entrées
  • ✅ Utiliser des ORM
  • ✅ Limiter les permissions de base de données

🔗 Man-in-the-Middle (MITM)

Interception de communications entre deux parties.

Techniques :

  • ARP Spoofing : Usurper l'adresse ARP
  • DNS Spoofing : Rediriger vers un faux site
  • SSL Stripping : Forcer HTTPS -> HTTP
  • Packet Sniffing : Capturer les paquets

Exemple d'attaque :

  1. L'attaquant se positionne entre le client et le serveur
  2. Le client envoie des données pensant les envoyer au serveur
  3. L'attaquant lit, modifie et transmet les données
  4. Le serveur reçoit les données modifiées

Défense :

  • ✅ Utiliser HTTPS/TLS
  • ✅ Vérifier les certificats
  • ✅ Utiliser un VPN
  • ✅ Éviter les réseaux WiFi publics

🎣 Phishing avancé

Faux emails et sites web pour voler les identifiants.

Techniques :

  • Spear Phishing : Phishing ciblé (personnalisé)
  • Whaling : Cibler les cadres supérieurs
  • Clone Phishing : Cloner un vrai email
  • SMTP spoofing : Falsifier l'adresse email

🚫 Contournement d'authentification

Contourner les mécanismes de sécurité d'authentification.

Techniques :

  • Default credentials : Utiliser les identifiants par défaut
  • Credential stuffing : Utiliser des listes d'identifiants compromis
  • Session hijacking : Voler les cookies de session
  • Token replay : Réutiliser un token valide

🐚 Reverse Shell

Obtenir un accès interactif à un système distant.

Comment ça marche :

  1. L'attaquant exécute un payload sur la cible
  2. La cible établit une connexion de retour à l'attaquant
  3. L'attaquant obtient un shell interactif
  4. L'attaquant peut exécuter des commandes

⬆️ Privilege Escalation

Augmenter les droits d'accès après une intrusion initiale.

Types :

  • Vertical escalation : D'utilisateur normal à administrateur
  • Horizontal escalation : D'un utilisateur à un autre au même niveau

Techniques :

  • Exploitation de kernel vulnerabilities
  • Mauvaises permissions de fichiers
  • Sudo misconfiguration
  • SUID binaries
  • Tasks/Cron jobs

🎭 Spoofing

Usurper l'identité de quelqu'un d'autre.

Types :

  • IP Spoofing : Usurper une adresse IP
  • MAC Spoofing : Changer l'adresse MAC
  • Email Spoofing : Falsifier l'adresse email
  • DNS Spoofing : Rediriger vers un faux serveur

🔙 Backdoors

Portes d'accès secrètes pour revenir plus tard.

Types :

  • Comptes d'utilisateur cachés
  • Web shells
  • Rootkits
  • Trojan horses

📊 Distributed Denial of Service (DDoS)

Surcharger un service pour le rendre inaccessible.

Types :

  • Volumetric : Attaque par bande passante massive
  • Protocol : Exploiter les failles de protocole
  • Application : Cibler l'application elle-même

⚠️ IMPORTANT

Ces techniques ne doivent être utilisées que dans un contexte légal et éthique !

  • ❌ Utiliser ces techniques sans permission = Criminel
  • ✅ Les utiliser avec permission = Professionnel
